# 📌 Project Overview

Smart City Road Accident Analytics is an **Advanced Python Data Analytics Project** that demonstrates a complete data analytics workflow using Python and its core data analysis libraries.

The project analyzes road accident data through a structured pipeline consisting of:

- Data Loading
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Professional Data Visualization

Unlike many analytics projects that rely on publicly available datasets, this project uses a **custom synthetic dataset generated using IRAAD (Indian Road Accident Analytics Dataset Generator)**, making the project completely self-contained and suitable for educational and analytical purposes.

---

# 🚗 About IRAAD Dataset

The dataset used in this project has been generated using **IRAAD (Indian Road Accident Analytics Dataset Generator)**.

IRAAD is a custom-built dataset generator designed to simulate realistic Indian road accident scenarios while maintaining logical consistency and analytical value.

The generated dataset represents multiple factors involved in road accidents, including:

- Accident Details
- Driver Information
- Vehicle Information
- Road Conditions
- Weather Conditions
- Emergency Response
- Risk Assessment

Dataset Information

| Property | Value |
|----------|-------|
| Dataset Type | Synthetic |
| Dataset Generator | IRAAD |
| Records | 50,000 |
| Columns | 61 |
| Format | CSV |

The dataset has been created solely for educational, research, and data analytics purposes.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Shailendrapathak878/Smart-City-Road-Accident-Analytics.git
```

Move to the project directory

```bash
cd Smart-City-Road-Accident-Analytics
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Run the project

```bash
cd src
python main.py
```
